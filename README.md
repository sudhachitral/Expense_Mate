# Expense Tracker Web App

A simple and clean **Expense Tracker Web Application** built using **Python (Flask)**, HTML, CSS, and SQLite.  
This app lets you add, edit, delete, and categorize expenses with a responsive UI and Dark/Light mode toggle.
